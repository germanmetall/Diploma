### Diploma

Contains two folders:

* CMS - for backend

* front - for frontend:

    * front-landing - for landing

    * front-platform - for platform

This is one git repository.

## To access CMS

Do this:

`cd CMS`

`npm run develop`

## To access landing

Do this:

`cd front/front-landing`

`npm run serve`

## To access platform

Do this:

`cd front/front-platform`

`npm run serve`
